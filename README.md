# pivx-downloader

## Usage

```

Usage: pivx-downloader [--os-flag]

Operating System Flags:

  --arch64    | Arch 64-bit
  --arm32     | Arm 32-bit
  --linux32   | Linux 32-bit
  --linux64   | Linux 64-bit
  --MacOS-dmg | MacOS DMG
  --macOS-tgz | MacOS Tarball
  --macOS-HSO | MacOS High Sierra Only DMG
  --win32-exe | Win32 Installer (Unsigned)
  --win32-zip | Win32 Zip
  --win64-exe | Win64 Installer (Unsigned)
  --win64-zip | Win64 Zip

Additional Flags:

  --help      | Displays this output

```
